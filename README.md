## Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/)
- [Rust](https://www.rust-lang.org/)

### Installation

1. Clone the repository
```bash
git clone [your-repo-url]
cd finder
```

2. Install dependencies
```bash
npm install
```

3. Run in development mode
```bash
npm run tauri dev
```

4. Build for production
```bash
npm run tauri build
```

## Tech Stack

- **Frontend**: SvelteKit + TypeScript
- **Backend**: Rust + Tauri
- **Build Tool**: Vite
- **Package Manager**: npm
